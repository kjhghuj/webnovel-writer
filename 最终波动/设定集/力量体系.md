# 力量体系设定

## 体系公理

### 核心信条/定律
1. **序列决定上限**：序列波动等级决定可适配机甲的最高等级
2. **能量守恒**：越强的攻击消耗越多能源
3. **共振法则**：驾驶员与机甲序列频率越接近，适配度越高，战力发挥越强
4. **无序特权**：唯有"无序波动"可突破序列限制，适配任何机体
5. **机甲驾驶五维要求**：机甲等级越高，对驾驶员的体能、智力、反应力、精神强度、操作技术要求越严苛，任一维度不达标将导致无法驾驶或遭受反噬

#### 特殊序列要求（X级/EX级/FX级）
X级以上机甲不仅需要五维达标，还需要额外苛刻条件：

| 等级 | 额外要求 | 失败后果 |
|------|----------|----------|

| **FX级** | 需要波纹共鸣——驾驶员的序列波动必须与机甲核心频率完全同步，形成"人甲共鸣" | 共鸣失败将引发能量暴走，驾驶员和机甲同时爆炸 |

### 反噬机制
当驾驶员任一维度不达标却强行驾驶时，将触发反噬：

| 不达标维度 | 反噬表现 | 后果 |
|------------|----------|------|
| **体能不足** | 过载压力导致骨骼碎裂、内脏破裂 | 重伤或死亡 |
| **智力不足** | 无法理解战术系统，操作混乱 | 机甲失控，误伤友军 |
| **反应力不足** | 无法及时响应战场变化 | 被敌人击中，机毁人亡 |
| **精神强度不足** | 精神崩溃，被机甲AI反噬 | 成为植物人或疯子 |
| **操作技术不足** | 误操作触发安全机制 | 机甲强制停机，任人宰割 |

### 公平性原则
- **普通人**：序列等级天生决定，无法突破
- **天才**：可通过训练提升小境界，大境界需机缘
- **主角破格**：无序波动能适配任何机体，无使用限制和副作用
- **反派破格**：莫森博士通过外星科技强行提升，代价是人性丧失

## 能力来源

### 能量/资源来源
- **序列波动**：基因觉醒，天生决定（主角除外）
- **灵能**：精神力量，可通过训练提升
- **机甲能源**：能源核心（掉落/合成/交易）

### 训练/修炼方法
| 体系 | 训练方式 | 提升速度 | 上限 |
|------|----------|----------|------|
| **序列波动** | 实战/生死搏杀 | 慢 | 天生决定 |
| **灵能** | 冥想/精神冲击 | 中 | 宗师级 |
| **机甲驾驶** | 模拟器/实战 | 快 | 受序列限制 |

### 社会控制机制
- **军方**：机甲学院，控制高级机甲分配
- **科学家组织**：序列研究所，垄断觉醒技术
- **聚居地**：贡献点制度，控制资源流动

## S级以上机甲特殊要求

### S级机甲（战略级）
**基础要求**：全部五维达到宗师级

**特殊准入条件**：
1. **机甲考验**：必须通过机甲自带的"试炼空间"（虚拟现实战斗，失败率90%）
2. **血脉纯度**：要求特定基因序列（天生决定，无法后天改变）
3. **序列匹配度**：与机甲核心频率匹配度≥85%（低于此值无法启动）
4. **精神共鸣**：需与机甲AI建立精神链接（建立过程痛苦异常，失败者精神受创）

**反噬风险**：
- 任何一维不达标：基因崩溃，寿命缩短50%
- 考验失败：精神创伤，可能永久无法驾驶机甲
- 匹配度不足：机甲自毁，驾驶员当场死亡

### X级机甲（实验级）
**基础要求**：全部五维达到传奇级，且突破人类极限

**特殊准入条件**：
1. **远古试炼**：必须通过上古遗迹中的"传承考验"（死亡率70%）
2. **血脉觉醒**：需要觉醒远古血脉（千年难遇）
3. **命运匹配**：机甲会自主选择驾驶员（拒绝者爆体而亡）
4. **精神融合**：需与机甲完全融合，人甲一体（融合过程持续72小时，痛苦不堪）

**反噬风险**：
- 任何一维不达标：直接爆体而亡
- 试炼失败：灵魂被机甲吞噬，永世不得超生
- 被机甲拒绝：全身经脉尽断，成为废人

### EX级机甲（专属级）
**基础要求**：全部五维达到传奇巅峰，与机甲形成共生关系

**特殊准入条件**：
1. **绑定仪式**：需与机甲进行血契绑定（一生只能绑定一台，解绑即死）
2. **共同成长**：机甲与驾驶员必须同步进化，一方停滞另一方也会受限
3. **灵魂印记**：驾驶员的灵魂会被机甲刻印（死后灵魂被机甲吸收）
4. **唯一性**：一台EX级机甲一生只认一个主人，他人无法驾驶

**反噬风险**：
- 绑定失败：灵魂破碎，当场死亡
- 成长不同步：双方互相拖累，实力倒退
- 强行解绑：灵魂被撕裂，永世不得超生

### FX级机甲（幻想级）
**基础要求**：超越人类极限，达到神话级

**特殊准入条件**：
1. **神之考验**：必须通过"观察者"设下的终极试炼（死亡率99%）
2. **宇宙认可**：需要得到宇宙法则的认可（亿万年难遇）
3. **因果绑定**：与机甲绑定后，命运与宇宙相连（一荣俱荣，一损俱损）
4. **永恒契约**：绑定后获得永恒生命，但失去死亡的权利

**反噬风险**：
- 任何不达标：被宇宙法则抹除存在
- 考验失败：灵魂被放逐到虚空中，永世游荡
- 被法则拒绝：因果崩坏，从时间线上被抹除

## 主角特权说明

### 无序波动的特殊性
主角的"无序波动"可无视上述所有要求：
- **无视体能限制**：再高的加速度也不会受伤
- **无视智力要求**：系统自动辅助战术分析
- **无视反应力门槛**：无序波动自动同步机甲动作
- **无视精神强度**：不会遭受任何精神反噬
- **无视操作技术**：无需训练即可发挥100%性能
- **无视血脉匹配**：任何机甲都可强行适配
- **无视考验限制**：无需试炼即可驾驶高级机甲

**迪化效果**：旁人无法理解主角为何能驾驶高级机甲，只能脑补他是"隐藏的天才""远古血脉觉醒者""机甲选中之人"等。

### 资源要求
| 大境界 | 所需资源 | 获取难度 |
|--------|----------|----------|
| G→F | 基础训练 +10 场战斗 | 低 |
| F→E | 能源核心×5+20 场战斗 | 中 |
| E→D | 高级营养剂×10+30 场战斗 | 中 |
| D→C | 稀有矿物×20+50 场战斗 | 高 |
| C→B | S 级异兽核心×1+100 场战斗 | 高 |
| B→A | 外星科技辅助×1+200 场战斗 | 极高 |
| A→S | 特殊机缘（生死突破） | 传说 |
| S→X/EX/FX | 特殊条件（血脉/绑定） | 唯一 |

### 突破方式
- **小境界**（G-→G→G+）：训练 + 实战积累
- **大境界**（G+→F-）：需要资源 + 生死突破
- **特殊序列**：需要特殊条件（主角：无序波动；其他人：血脉/外星科技）

### 失败代价
- **小境界失败**：停滞 1-3 个月，无副作用
- **大境界失败**：序列受损，降级或永久停滞
- **特殊序列失败**：死亡或基因崩溃

### 越级代价
- **越 1 级战斗**：消耗翻倍，可短暂爆发
- **越 3 级战斗**：燃烧寿命，每次缩短 1 年
- **越 5 级+**：必死，除非有外援

## 战斗规则

### 伤害与防御逻辑
```
攻击力 = 机甲基础攻击 × 适配度% × 技能加成
防御力 = 机甲装甲 × 灵能护盾% × 闪避率
最终伤害 = 攻击力 - 防御力 × 克制系数
```

### 战斗节奏特点
- **G-F 级**：近身肉搏，消耗战，拼耐力
- **E-C 级**：中距离游击，拼反应和预判
- **B-A 级**：灵能对抗，拼精神力和领域
- **S 级+**：战略对决，一人可影响战局

### 克制/反制关系
| 攻击类型 | 克制 | 被克 |
|----------|------|------|
| **近战** | 远程（近身后） | 远程（拉开距离） |
| **远程** | 近战（保持距离） | 灵能（可拦截） |
| **灵能** | 精神防御低者 | 灵能防御高者 |
| **范围** | 群体目标 | 单体高防 |
| **单体** | 高防目标 | 范围消耗 |

### 逃生/撤退机制
- **G-C 级**：依靠机甲速度，硬突破
- **B-A 级**：灵能干扰 + 烟雾弹
- **S 级+**：空间跳跃（需特殊装备）
- **主角特权**：无序波动可短暂"隐身"序列波动

## 体系漏洞与利用

### 漏洞（必须可解释）
1. **适配度漏洞**：理论上序列决定适配，但主角无序波动可绕过
2. **灵能体系**：独立于序列波动，可后天训练，是普通人唯一出路
3. **共振频率**：相同序列不同频率，可 exploited 进行"频率干扰"

### 主角如何利用
- **无序适配**：可驾驶任何机甲，收集情报更高效
- **双修优势**：序列 + 灵能同时提升，战力翻倍
- **频率伪装**：可模拟任何序列频率，隐藏身份

### 反派如何反制
- **莫森博士**：外星科技强行提升，无视序列限制
- **军方**：人海战术，消耗主角
- **清理者**：外星科技压制，针对性封锁
