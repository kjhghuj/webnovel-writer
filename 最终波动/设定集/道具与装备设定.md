# 武器装备

## 近战武器

### 高能震荡刀
| 属性 | 说明 |
|------|------|
| **类型** | 近战武器 |
| **等级** | G-S级 |
| **原理** | 高频震荡产生切割力 |
| **伤害** | 可切割异兽装甲（根据等级） |
| **耗能** | 低（G级）-高（S级） |
| **来源** | 人类工业制造 |

**等级划分**：
- G级：基础震荡，切割F级以下异兽
- F-E级：强化震荡，切割D级以下异兽
- D-C级：高能震荡，切割B级以下异兽
- B-A级：超频震荡，切割A级以下异兽
- S级：极限震荡，可切割S级异兽外壳

### 灵能实体剑
| 属性 | 说明 |
|------|------|
| **类型** | 近战武器（灵能系） |
| **等级** | C-EX级 |
| **原理** | 灵能凝聚成实体剑刃 |
| **伤害** | 物理+精神双重伤害 |
| **耗能** | 精神力持续消耗 |
| **来源** | 灵能者专属 |

**说明**：只有灵能达到进阶（C级）以上才能使用，剑刃颜色随灵能属性变化。

# 道具获取途径

| 道具类型 | 主要来源 | 次要来源 | 黑市流通 |

| 稀有矿物 | 红区采集 | 交易 | 有 |
| 探测设备 | 军方/组织 | 黑市 | 有限 |
| 伪装道具 | 黑市 | 自制 | 唯一来源 |
| 外星科技 | 禁区探索 | 莫森组织 | 极少 |
