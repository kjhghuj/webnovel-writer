# 主角：白岚

## 基础形象

### 容貌
- **脸型**：棱角分明的瓜子脸，下颌线条锐利但不突兀
- **眼睛**：深邃的黑瞳，眼尾微微上挑，眼神冷静而疏离
- **眉毛**：剑眉，眉峰清晰，不怒自威
- **鼻子**：挺直的鼻梁，鼻头略尖
- **嘴唇**：薄唇，嘴角习惯性微微下垂，显得冷淡
- **肤色**：偏白，但非病态苍白，透着健康光泽
- **发型**：黑色短发，刘海略长遮住部分额头，发尾微微翘起
- **特殊标记**：后颈有一道浅浅的疤痕（穿越实验留下的痕迹）

### 身材
- **身高**：182cm（穿越后）
- **体重**：75kg，精瘦型
- **体型**：宽肩窄腰，肌肉线条分明但不夸张
- **手部**：修长有力，指节分明，常年驾驶机甲留下薄茧
- **姿态**：习惯性微微低头，眼神从下方看人，显得警惕

### 气质
- **整体印象**：冷峻、疏离、难以接近
- **眼神特质**：看人时先打量再开口，习惯性保持安全距离
- **气场**：不怒自威，不说话时给人压迫感
- **微表情**：思考时会微微眯眼，做决定时会抿唇

### 标志性特征
- **穿着风格**：深色系为主，实用主义，偏好带帽外套（方便隐藏身份）
- **随身物品**：永远带着一副黑色战术手套（掩盖手部识别特征）
- **习惯动作**：说话时习惯用左手轻敲桌面或大腿（思考节奏）

# 女主一：苏晚晴

## 基础形象

### 容貌
- **脸型**：鹅蛋脸，线条柔和精致
- **眼睛**：淡褐色杏眼，戴一副细框眼镜，镜片后眼神专注而清冷
- **眉毛**：柳叶眉，眉尾略微下垂，显得温婉
- **鼻子**：小巧挺直，鼻翼略窄
- **嘴唇**：樱桃小嘴，唇色偏淡，习惯性轻抿
- **肤色**：白皙，常年实验室工作略显苍白
- **发型**：深棕色长发，习惯性扎成低马尾，刘海整齐地梳向一侧
- **特殊标记**：右手食指有长期实验留下的细小疤痕

### 身材
- **身高**：168cm
- **体重**：52kg，纤细型
- **体型**：苗条纤细，腰肢盈盈一握
- **手部**：手指修长白皙，指甲修剪整齐，常沾有机油或试剂痕迹
- **姿态**：习惯性微微前倾（看显微镜养成的习惯），走路步伐小而快

### 气质
- **整体印象**：知性、清冷、书卷气
- **眼神特质**：专注时眼神锐利，平时温和疏离
- **气场**：安静但不容忽视，专业素养带来的自信
- **微表情**：思考时会推眼镜，紧张时会无意识地绞手指

### 标志性特征
- **穿着风格**：白色或浅灰色研究服，里面搭配高领毛衣
- **随身物品**：永远带着一个小型工具包和数据平板
- **习惯动作**：推眼镜、记录数据时的专注侧脸

# 女主二：雷玥

## 基础形象

### 容貌
- **脸型**：方脸，轮廓硬朗，带着英气
- **眼睛**：琥珀色虎目，眼神锐利如刀，充满战意
- **眉毛**：粗黑的剑眉，眉峰上扬，显得英气逼人
- **鼻子**：高挺的鼻梁，略带鹰钩
- **嘴唇**：薄唇，嘴角习惯性上扬（挑衅的笑）
- **肤色**：健康的小麦色，阳光下有光泽
- **发型**：利落的短发，染成深栗色，发梢微卷
- **特殊标记**：左脸颊有一道浅浅的战斗疤痕（荣耀的象征）

### 身材
- **身高**：175cm
- **体重**：62kg，健美型
- **体型**：肌肉线条分明，肩宽腰细，充满力量感
- **手部**：手掌宽大，指关节突出，布满老茧
- **姿态**：站如松坐如钟，习惯性双手抱胸或叉腰

### 气质
- **整体印象**：豪爽、霸气、女中豪杰
- **眼神特质**：直视对方，从不回避，充满侵略性
- **气场**：强势、自信、让人不敢轻视
- **微表情**：战斗兴奋时眼睛会发光，愤怒时会咬紧牙关

### 标志性特征
- **穿着风格**：黑色或深灰色战斗服，偏好露出手臂的设计
- **随身物品**：总是带着一把战术匕首（哥哥遗物）
- **习惯动作**：活动手腕关节（热身动作），甩头发

# 女主三：沈若雪

## 基础形象

### 容貌
- **脸型**：瓜子脸，线条精致优雅
- **眼睛**：冰蓝色凤眼，眼神冷静克制，带着军人特有的锐利
- **眉毛**：细长平直的眉毛，显得端庄
- **鼻子**：挺直精致的鼻梁
- **嘴唇**：薄唇，嘴角习惯性紧绷，显得严肃
- **肤色**：白皙，但不同于苏晚晴的书卷白，而是养尊处优的细腻
- **发型**：银灰色长发（染的），习惯性盘成髻，露出优雅的颈部线条
- **特殊标记**：右肩有军方精英标志的纹身

### 身材
- **身高**：170cm
- **体重**：55kg，匀称型
- **体型**：身材匀称，线条优美，军人的干练与女性的柔美结合
- **手部**：手指修长，指甲修剪得整整齐齐，保持军人仪容
- **姿态**：标准的军人姿态，挺胸抬头，步伐稳健

### 气质
- **整体印象**：冷艳、优雅、军人气质
- **眼神特质**：冷静克制，但深处藏着温度
- **气场**：距离感，但让人心生敬意
- **微表情**：犹豫时会轻咬下唇，做决定时眼神坚定

### 标志性特征
- **穿着风格**：军方制服，永远熨烫得整整齐齐，勋章佩戴规范
- **随身物品**：军牌、配枪、战术手表
- **习惯动作**：敬礼、整理衣领、看手表

- **鼻子**：鹰钩鼻，鼻头略红

### 身材

# 观察者（人类形态投影）

## 基础形象

### 容貌
- **脸型**：没有固定面容，每次出现时都不同，但都是完美的人类面容
- **眼睛**：纯银色瞳孔，没有眼白，仿佛两个银色的球体
- **其他特征**：没有毛发，皮肤呈金属光泽

### 身材
- **身高**：可随意变化，通常呈现190cm的修长身材
- **体型**：完美比例，但透着非人的冰冷

### 气质
- **整体印象**：绝对理智、非人、高高在上
- **气场**：让人本能地想要臣服
